http GET /HelloWorld/{name} （参数 date:String） - 测试用接口

http POST /HelloWorld/{name} （参数 date:String） - 测试用接口

1. http POST /sign_up （参数 schoolID, account, password） - 注册

2. http POST /sign_in （参数 account, password） - 登录

3. http POST /account/{account} （参数 userToken, name, info, contact） - 设置个人信息

4. http GET /account/{account} （参数 userToken, account） - 获取个人信息

5. http POST /course/add （参数 userToken, account, courseName, courseInfo, coursePlan） - 教师开课

6. http GET /course/manage （参数 userToken） - 管理课程（列出课程）

7. http GET /course/search （参数 keyword） - 搜索课程（列出课程）

8. http GET /course/detail/{courseID} - 课程细节

9. http POST /course/select/{courseID} （参数 userToken, account） - 选课

10. http POST /upload_material （参数 userToken, account, courseID, file） - 上传资源

11. http GET /material （参数 courseID, userToken, account） - 获取资源

12. http POST /course/evaluation/{courseID} （参数 userToken, account, score, comment） - 评教

13. http GET /course/evaluation/{courseID} （参数 userToken, account） - 获取评教信息
